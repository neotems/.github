## 🚀 关于我们 (About Us)

**智核九章** 是一家专注于 **人工智能底层技术与前沿应用** 的创新组织。

我们的名字源于两重寓意：
- **智核 (Core Intelligence)**：致力于构建智能系统的核心引擎，探索算法的边界与效率。
- **九章 (Nine Chapters)**：致敬《九章算术》的算法源头精神，同时象征对技术极致（九为极数）的追求。

我们相信，最好的 AI 应当是 **逻辑严密如古算，算力澎湃如量子**。
